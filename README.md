# CodSoft Python Internship Tasks

This repository contains the three Python projects that I completed during my CodSoft Python Internship. These projects helped me improve my Python programming skills and understand how to build simple applications using Tkinter.

## Projects

### 1. To-Do List Application
This application helps users manage their daily tasks. Users can add, edit, delete, and mark tasks as completed. The tasks are saved automatically so they are available the next time the application is opened.

**Features**
- Add Task
- Edit Task
- Delete Task
- Mark Task as Completed
- Save Tasks
- Simple and Easy-to-Use Interface

**Tools Used**
- Python
- Tkinter
- JSON

---

### 2. Calculator
This is a simple calculator built using Python. It performs basic arithmetic operations through an easy-to-use graphical interface.

**Features**
- Addition
- Subtraction
- Multiplication
- Division
- Clear Screen

**Tools Used**
- Python
- Tkinter

---

### 3. Password Generator
This application generates strong and secure passwords based on the length entered by the user. The generated password contains letters, numbers, and special characters.

**Features**
- User-defined Password Length
- Random Password Generation
- Strong and Secure Passwords
- Easy-to-Use Interface

**Tools Used**
- Python
- Tkinter

## How to Run

1. Download or clone this repository.
2. Open the required project folder.
3. Run the Python file using:

```bash
python app.py
```

or

```bash
python calculator.py
```

or

```bash
python password.py
```
